### **6.1. –ß—Ç–æ —Ç–∞–∫–æ–µ CRUD –∏ –∑–∞—á–µ–º –æ–Ω –Ω—É–∂–µ–Ω?**
CRUD (Create, Read, Update, Delete) ‚Äî —ç—Ç–æ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–π –Ω–∞–±–æ—Ä –æ–ø–µ—Ä–∞—Ü–∏–π –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å –±–∞–∑–∞–º–∏ –¥–∞–Ω–Ω—ã—Ö. –û–Ω –Ω–µ–æ–±—Ö–æ–¥–∏–º –¥–ª—è —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è –∑–∞–ø–∏—Å—è–º–∏, –∏—Ö —Å–æ–∑–¥–∞–Ω–∏—è, —á—Ç–µ–Ω–∏—è, –æ–±–Ω–æ–≤–ª–µ–Ω–∏—è –∏ —É–¥–∞–ª–µ–Ω–∏—è. –≠—Ç–∏ –æ–ø–µ—Ä–∞—Ü–∏–∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –≤–æ –≤—Å–µ—Ö –≤–µ–±-–ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è—Ö –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è –∏ –æ–±—Ä–∞–±–æ—Ç–∫–∏ –¥–∞–Ω–Ω—ã—Ö.

### **6.2. –†–µ–∞–ª–∏–∑–∞—Ü–∏—è CRUD –≤ FastAPI**

–í FastAPI –º—ã —Ä–µ–∞–ª–∏–∑—É–µ–º CRUD —á–µ—Ä–µ–∑ SQLAlchemy ORM. –≠—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –Ω–∞–º —Ä–∞–±–æ—Ç–∞—Ç—å —Å –±–∞–∑–æ–π –¥–∞–Ω–Ω—ã—Ö –≤ –æ–±—ä–µ–∫—Ç–Ω–æ-–æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–º —Å—Ç–∏–ª–µ –∏ –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –≤—ã–ø–æ–ª–Ω—è—Ç—å SQL-–∑–∞–ø—Ä–æ—Å—ã –±–µ–∑ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏ –ø–∏—Å–∞—Ç—å –∏—Ö –≤—Ä—É—á–Ω—É—é.

–°–æ–∑–¥–∞—ë–º —Ñ–∞–π–ª `routers/books.py`:

```python
from fastapi import APIRouter, Depends, HTTPException
from sqlalchemy.orm import Session
from database import get_db
from models import BookDB
from schemas import BookCreate, BookResponse

router = APIRouter()

# –û–ø–µ—Ä–∞—Ü–∏—è CREATE (–î–æ–±–∞–≤–ª–µ–Ω–∏–µ –Ω–æ–≤–æ–π –∫–Ω–∏–≥–∏ –≤ –±–∞–∑—É –¥–∞–Ω–Ω—ã—Ö)
@router.post("/", response_model=BookResponse)
def create_book(book: BookCreate, db: Session = Depends(get_db)):
    """
    –°–æ–∑–¥–∞–µ—Ç –Ω–æ–≤—É—é –∫–Ω–∏–≥—É –≤ –±–∞–∑–µ –¥–∞–Ω–Ω—ã—Ö.
    """
    db_book = BookDB(**book.dict())  # –ü—Ä–µ–æ–±—Ä–∞–∑—É–µ–º –¥–∞–Ω–Ω—ã–µ Pydantic-–º–æ–¥–µ–ª–∏ –≤ SQLAlchemy-–º–æ–¥–µ–ª—å
    db.add(db_book)  # –î–æ–±–∞–≤–ª—è–µ–º –∫–Ω–∏–≥—É –≤ —Å–µ—Å—Å–∏—é
    db.commit()  # –§–∏–∫—Å–∏—Ä—É–µ–º –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤ –±–∞–∑–µ –¥–∞–Ω–Ω—ã—Ö
    db.refresh(db_book)  # –û–±–Ω–æ–≤–ª—è–µ–º –æ–±—ä–µ–∫—Ç, —á—Ç–æ–±—ã –ø–æ–ª—É—á–∏—Ç—å –ø—Ä–∏—Å–≤–æ–µ–Ω–Ω—ã–π ID
    return db_book

# –û–ø–µ—Ä–∞—Ü–∏—è READ (–ü–æ–ª—É—á–µ–Ω–∏–µ —Å–ø–∏—Å–∫–∞ –≤—Å–µ—Ö –∫–Ω–∏–≥)
@router.get("/")
def get_books(db: Session = Depends(get_db)):
    """
    –ü–æ–ª—É—á–∞–µ—Ç —Å–ø–∏—Å–æ–∫ –≤—Å–µ—Ö –∫–Ω–∏–≥ –≤ –±–∞–∑–µ –¥–∞–Ω–Ω—ã—Ö.
    """
    return db.query(BookDB).all()

# –û–ø–µ—Ä–∞—Ü–∏—è READ (–ü–æ–ª—É—á–µ–Ω–∏–µ –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–π –∫–Ω–∏–≥–∏ –ø–æ ID)
@router.get("/{book_id}")
def get_book(book_id: int, db: Session = Depends(get_db)):
    """
    –ü–æ–ª—É—á–∞–µ—Ç –∫–Ω–∏–≥—É –ø–æ –µ—ë ID.
    """
    book = db.query(BookDB).filter(BookDB.id == book_id).first()
    if not book:
        raise HTTPException(status_code=404, detail="Book not found")
    return book

# –û–ø–µ—Ä–∞—Ü–∏—è UPDATE (–û–±–Ω–æ–≤–ª–µ–Ω–∏–µ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏ –æ –∫–Ω–∏–≥–µ)
@router.put("/{book_id}")
def update_book(book_id: int, updated_book: BookCreate, db: Session = Depends(get_db)):
    """
    –û–±–Ω–æ–≤–ª—è–µ—Ç –¥–∞–Ω–Ω—ã–µ –∫–Ω–∏–≥–∏ –ø–æ –µ—ë ID.
    """
    book = db.query(BookDB).filter(BookDB.id == book_id).first()
    if not book:
        raise HTTPException(status_code=404, detail="Book not found")
    for key, value in updated_book.dict().items():
        setattr(book, key, value)  # –û–±–Ω–æ–≤–ª—è–µ–º –ø–æ–ª—è –∫–Ω–∏–≥–∏
    db.commit()
    db.refresh(book)
    return book

# –û–ø–µ—Ä–∞—Ü–∏—è DELETE (–£–¥–∞–ª–µ–Ω–∏–µ –∫–Ω–∏–≥–∏ –ø–æ ID)
@router.delete("/{book_id}")
def delete_book(book_id: int, db: Session = Depends(get_db)):
    """
    –£–¥–∞–ª—è–µ—Ç –∫–Ω–∏–≥—É –∏–∑ –±–∞–∑—ã –¥–∞–Ω–Ω—ã—Ö –ø–æ –µ—ë ID.
    """
    book = db.query(BookDB).filter(BookDB.id == book_id).first()
    if not book:
        raise HTTPException(status_code=404, detail="Book not found")
    db.delete(book)
    db.commit()
    return {"message": "Book deleted successfully"}
```
